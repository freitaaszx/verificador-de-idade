# 🧾 Verificador de Idade em C#

Este é um projeto simples em C# que solicita o nome e a idade do usuário e verifica se ele pode se alistar no serviço militar.

---

## 🚀 Funcionalidades

- Solicita o nome do usuário
- Solicita a idade do usuário
- Verifica se a idade é maior ou igual a 18 anos
- Exibe uma mensagem informando se pode ou não se alistar

---

## 🧠 Lógica do programa

- Se `idade >= 18` → pode se alistar
- Se `idade < 18` → não pode se alistar
